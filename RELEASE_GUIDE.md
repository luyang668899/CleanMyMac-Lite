# CleanMyMac Lite 发布指南

本指南详细说明如何构建和使用 CleanMyMac Lite 工具，以便用户可以直接下载和使用。

## 项目概述

CleanMyMac Lite 是一款功能强大的 macOS 系统清理工具，具有以下核心功能：

- 深度扫描清理：针对常见应用逐一定制扫描方案
- 大文件清理：查找占用超过50M的大文件
- 重复文件清理：智能比对重复内容
- 相似照片清理：极速筛选相似照片
- 浏览器隐私清理：保护浏览器隐私
- 应用卸载：清理应用更干净，不怕卸载有残留
- 开启启动项管理：管理应用开机自启动
- 自定义状态栏展示信息：实时显示系统状态

## 新增功能模块

我们已经开发并集成了以下新的清理功能模块：

1. **重复文件清理器** (`LemonDuplicateFileCleaner`)：扫描并清理重复的文件，释放存储空间
2. **缓存清理器** (`LemonCacheCleaner`)：清理系统和应用的缓存文件
3. **日志文件清理器** (`LemonLogCleaner`)：清理系统和应用的日志文件
4. **临时文件清理器** (`LemonTempFileCleaner`)：清理系统临时目录的文件
5. **应用卸载残留清理器** (`LemonAppResidualCleaner`)：清理已卸载应用的残留文件

## 构建指南

### 前提条件

- macOS 10.11 或更高版本
- Xcode 10.0 或更高版本
- CocoaPods 1.8.0 或更高版本

### 构建步骤

1. **克隆项目**

   ```bash
   git clone https://github.com/yourusername/CleanMyMac-Lite.git
   cd CleanMyMac-Lite
   ```

2. **安装依赖**

   ```bash
   pod install
   ```

3. **打开工作空间**

   ```bash
   open Lemon.xcworkspace
   ```

4. **构建应用**

   - 在 Xcode 中，选择 `Product` > `Archive`
   - 等待构建完成后，在 Organizer 中选择构建产物
   - 点击 `Distribute App`，选择 `Development` 或 `Ad Hoc`
   - 按照提示完成导出过程

5. **安装应用**

   - 导出完成后，将生成的 `.app` 文件拖放到 `/Applications` 目录
   - 或者使用项目中的 `copyLemonToApplication.sh` 脚本：

     ```bash
     ./Lemon/copyLemonToApplication.sh /path/to/Lemon.app
     ```

## 使用指南

### 首次启动

1. 打开 Applications 文件夹中的「CleanMyMac Lite」应用
2. 首次启动时，系统会提示授权访问权限，点击「允许」
3. 等待应用初始化完成

### 基本使用

1. **深度扫描**：点击主界面上的「扫描」按钮，应用会自动扫描系统中的垃圾文件
2. **清理文件**：扫描完成后，点击「清理」按钮，应用会自动清理选中的文件
3. **功能模块**：在左侧导航栏中选择不同的功能模块，如「大文件」、「重复文件」、「相似照片」等
4. **自定义设置**：点击顶部菜单栏中的「CleanMyMac Lite」>「偏好设置」，可自定义应用行为

### 新增功能使用

#### 重复文件清理
1. 在左侧导航栏中选择「重复文件」
2. 点击「开始扫描」按钮，选择要扫描的文件夹
3. 扫描完成后，查看重复文件列表，选择要删除的文件
4. 点击「清理」按钮，删除选中的重复文件

#### 缓存清理
1. 在左侧导航栏中选择「缓存清理」
2. 点击「开始扫描」按钮，应用会自动扫描系统和应用的缓存文件
3. 扫描完成后，查看缓存文件列表，选择要删除的文件
4. 点击「清理」按钮，删除选中的缓存文件

#### 日志文件清理
1. 在左侧导航栏中选择「日志文件」
2. 点击「开始扫描」按钮，应用会自动扫描系统和应用的日志文件
3. 扫描完成后，查看日志文件列表，选择要删除的文件
4. 点击「清理」按钮，删除选中的日志文件

#### 临时文件清理
1. 在左侧导航栏中选择「临时文件」
2. 点击「开始扫描」按钮，应用会自动扫描系统临时目录的文件
3. 扫描完成后，查看临时文件列表，选择要删除的文件
4. 点击「清理」按钮，删除选中的临时文件

#### 应用卸载残留清理
1. 在左侧导航栏中选择「应用卸载」
2. 选择已卸载的应用，点击「扫描残留」按钮
3. 扫描完成后，查看残留文件列表，选择要删除的文件
4. 点击「清理」按钮，删除选中的残留文件

## 安全注意事项

1. **系统文件保护**：应用会自动检测并保护系统关键文件，不会误删重要文件
2. **文件备份**：清理前建议备份重要文件，以防意外情况
3. **权限管理**：首次使用时请确保授予应用所需的权限，否则可能影响部分功能的正常使用
4. **定期更新**：定期更新应用以获取最新的清理规则和功能改进

## 故障排除

### 常见问题

1. **扫描速度慢**
   - 原因：扫描范围过大或硬盘速度较慢
   - 解决：缩小扫描范围，或在系统空闲时进行扫描

2. **清理后空间未释放**
   - 原因：可能是系统缓存或其他临时文件未完全清理
   - 解决：重启系统后再次扫描清理

3. **应用崩溃**
   - 原因：可能是权限问题或系统兼容性问题
   - 解决：确保授予应用所需的权限，或更新到最新版本
